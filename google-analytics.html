Getting setup with Google Search Console (GSC) and Analytics (GA) for GitHub Pages User Page: a brief walkthrough

Estimated time to complete: 5 to 30 minutes

Google Search Console ...

Hold your breath ...

  1. Sign into GSC
  2. Add property
    • https://temptemp3.github.io <= your site URL here
  3. Validate property
    • copy validation html file into html directory
    • deploy
      • see how to setup github pages for user super fast track
    • follow GSC instructions
  4. Fetch as Google
    • PC and Mobile
  5. done

※ Adding a property to your GitHubs Page is straightforward to the point that it doesn’t not require any changes to your templates to get set up.,

Google Analytics ...

This may take a while ...

  1. Create account
  2. Add tracking id code to template
  3. Deploy
  4. Check real-time reporting
  5. Done

※ If you do not see anything in the real-time report, do check you view setting and make sure that website URLs is the one using https, otherwise pageviews will not show up in the real-time report.

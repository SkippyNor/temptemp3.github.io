bash ssh without password

ssh login without password

1. use clipboard

cat p.txt > /dev/clipboard
ssh ${user}@${host} -p ${port} ${other_options} # paste on prompt

2. use sshpass

There is a command line utility called sshpass that allows you to login to a remote host through ssh. It requires installation and setting of environmental variables.

3. use key

The preferred method to connect through ssh without password. Omit -p option and supply -i identity in ssh command line.

Install requirements

Install php

http://php.net/manual/en/install.php

Install composer

https://getcomposer.org/download/

  • If using homestead

Install vagrant

https://www.vagrantup.com/downloads.html

You are setup to create laravel projects

Create new laravel project 

laravel new project-name

Install homestead

composer require laravel/composer --dev

Vagrant up

vagrant up

You want users to be able to login and register

Make auth using artisan

php artisan make:auth

If you see this

SQLSTATE[42S02]: Base table or view not found: 
1146 Table 'homestead.users' doesn't exist (42S02)

you need to vagrant ssh into homestead and run 

cd code
php artisan migrate

Users can login and register

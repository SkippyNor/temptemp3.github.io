bash check file permissions

Suppose that there is a file in your bash script that you want to check the file permissions for. Here is a quick snippet to accomplish the task using ls and cut.

_() { # 1 - path to file
  ls -s ${1} |
  cut '-d ' '-f1' # expect [-dl][-r][-w][-x][-r][-w][-x][-r][-w][-x]
}

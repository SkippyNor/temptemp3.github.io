what is a bash one-liner?

a bash one-liner?

  • a shell script snippet that can be executed on a single line

Joining commands by list operators a bash one-liner can be any bash script executable or library excluding interpreter directive lines and document here blocks.

not a bash one-liner

  • lone
    • function definitions
    • variable declarations

flush changes in git repository

one-liner
{ _() { { local message ; } ; git status ; echo "ok?" ; read ; echo "enter message:" ; read message ; git add . ; git commit -m "${message} $( date +%s )" ; } ; _ ; }
